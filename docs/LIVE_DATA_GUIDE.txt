Solution: That RPC might be down. System will use others.

   • Falls back to synthetic data if needed

Issue: Badge shows "🟡 DEMO MODE"
Solution: No blockchain connections established
   • Check internet connection
   • System works fine with synthetic data
   • All features still functional

Issue: Only seeing 🟡 SIM transactions
Solution: Blockchain RPCs might be rate-limited
   • Normal behavior during high traffic
   • Real data will appear when available
   • System automatically mixes in real txs

Issue: Port 8080 in use
Solution: Kill existing process
   lsof -ti:8080 | xargs kill -9
